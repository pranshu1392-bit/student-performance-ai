# 🎓 Student Performance & Risk Prediction System

## 📌 Project Description
This project predicts student academic performance based on factors like study hours, attendance, and assignment completion. It also classifies students into risk levels (Low, Medium, High) using a manually implemented regression model without external libraries.

---

## 🎯 Objectives
- Predict student marks using input factors
- Identify at-risk students
- Demonstrate machine learning concepts without using libraries
- Build a simple and user-friendly system

---

## ⚙️ Features
- 📊 Marks prediction
- ⚠️ Risk classification (Low / Medium / High)
- 👨‍🎓 Multiple student support
- 🖥️ Menu-driven program
- 🧠 ML logic implemented from scratch

---

## 🛠️ Technologies Used
- Python (Core Python only)
- No external libraries used
